## Usage

See the website to search for icons.

All icons are in the `/icons` folder of this repo. It follows a `{color}/{format}` structure:

```
├── icons
│   ├── black
│   │   ├── png
│   │   ├── svg
│   │   └── webp
│   └── white
│       ├── png
│       ├── svg
│       └── webp
```
